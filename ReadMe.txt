README: Exploratory Data Analysis Course Project Week 1 Assignment
Project Name:  Project Assignment

This is the homework assignment for the Exploratory Data Analysis, Week 1 Assignment.

Installation:

There are 4 R files:  plot1.R, plot2.R, plot3.R and plot4.R.  

Each R file is self contained and will download the required zipped datafile, unzip 
and generate the 4 graphic PNG files: plot1.png, plot2.png, plot3.png and plot4.png.
